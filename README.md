# Mobile-Weather-Station
Mobile Weather Station based on Arduino and NB-Iot. (SJTU course project) <br>
### 开发板选型：Arduino Nano
![](https://github.com/HollyHouHou/Mobile-Weather-Station/raw/master/nano引脚图.jpg)
### Modules
* NB75-B5 NB-Iot模块
* Arduino Nano
* GY-30 光照传感器
* GP2Y1014AU 粉尘传感器
* GY-BMP280 高度传感器
* OLED 显示器
* GUVA 紫外线传感器
* DHT22 温湿度传感器
